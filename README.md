# FaceFollowingRobot
This is a robot that determines the loaction of your face, and through some distance-based correction, turns and drives towards the person.  
I made this robot because I wanted to start expirementing with CV in robotics.  
The drivetrain utilizes 2 wheels with a motor on each wheel, and a bearing in the front to provide stability without restricting movement.  
The brains of the robot are a Seeed Studio XIAO ESP32 S3 Sense with a camera attached, which will run a SenseCraft face detection model.
